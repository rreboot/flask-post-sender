# Flask post sender

Рассылка новостей в сторонние сервисы

## Get started

* Create virtualenv `python3 -m venv venv`
* Activate it `. venv/bin/activate`
* Install requirements `pip install -r requirements.txt`
* Run `python manage.py runserver`
